**open_dispute.rs:**

The handler performs these steps:
1. **Validate caller is a market participant**: Require the provided `UserPosition` PDA belongs to the caller and has `yes_amount > 0 || no_amount > 0` (they placed at least one bet).
2. **Validate market state**: `market.state == 0` (still Open -- not already resolved or disputed).
3. **Validate grace period expired**: `clock.unix_timestamp > market.resolution_time + 172800` (48h grace has passed).
4. **Validate no MPC lock**: `!market.mpc_lock` (no in-flight computation).
5. **Validate enough resolvers**: `resolver_registry.resolvers.len() >= 7` (need at least 7 for jury).
6. **Select 7 jurors deterministically**: Use a pseudo-random seed derived from `market.id` XOR'd with `clock.slot` (or `clock.unix_timestamp`). Implement Fisher-Yates partial shuffle on the registry list to pick 7 unique resolvers. Store their pubkeys and stake snapshots (read from remaining_accounts Resolver PDAs).
7. **Create Dispute PDA** (init, payer = escalator): Set `status = 0` (Voting), `jurors`, `juror_stakes`, `votes_submitted = 0`, `vote_count = 0`, `quorum = 5`, `voting_start = clock.unix_timestamp`, `voting_end = clock.unix_timestamp + 172800` (48h voting window), `tiebreaker_added = false`, `escalator = caller`, `mpc_lock = false`.
8. **Create DisputeTally PDA** (init, payer = escalator): Set `market_id`, all encrypted fields to `[0u8; 32]`, `nonce = 0`. The init_dispute_tally MPC (Plan 03) will initialize with encrypted zeros.
9. **Transition market state**: `market.state = 3` (Disputed).
10. **Increment active_disputes** on each selected Resolver account (passed via remaining_accounts, validated by PDA seeds).

**Juror selection algorithm** (deterministic, on-chain):
```rust
// Pseudo-random seed from market ID + slot
let seed = market.id ^ (clock.slot as u64);
let n = resolver_registry.resolvers.len();
let mut selected_indices: Vec<usize> = Vec::with_capacity(7);

// Fisher-Yates partial selection (pick 7 from n)
let mut available: Vec<usize> = (0..n).collect();
for i in 0..7 {
    // Simple hash-based index: (seed + i as u64) % remaining
    let remaining = available.len();
    let idx = ((seed.wrapping_add(i as u64).wrapping_mul(6364136223846793005).wrapping_add(1)) % remaining as u64) as usize;
    selected_indices.push(available.remove(idx));
}
```
