    }

    **Accounts struct:**
    - `creator`: Signer, mut (pays rent for Market PDA + vault)
    - `config`: Account<Config>, mut (market_counter incremented), seeds = [b"config"], bump = config.bump, `constraint = !config.paused @ AvenirError::ProtocolPaused`
    - `whitelist`: Account<CreatorWhitelist>, seeds = [b"whitelist", creator.key().as_ref()], bump = whitelist.bump, `constraint = whitelist.active @ AvenirError::CreatorNotWhitelisted`
    - `market`: Account<Market>, init, payer = creator, space = 8 + Market::INIT_SPACE, seeds = [b"market", (config.market_counter + 1).to_le_bytes().as_ref()], bump
    - `market_vault`: Account<TokenAccount>, init, payer = creator, seeds = [b"vault", (config.market_counter + 1).to_le_bytes().as_ref()], bump, token::mint = usdc_mint, token::authority = market
    - `usdc_mint`: Account<Mint>, `constraint = usdc_mint.key() == config.usdc_mint @ AvenirError::InvalidMint`
    - `token_program`: Program<Token>
    - `system_program`: Program<System>

    **IMPORTANT:** For the `market` and `market_vault` seeds, the market_id is `config.market_counter + 1` (the NEXT id). This must be computed in the seeds expression. Use `#[instruction(params: CreateMarketParams)]` attribute if needed for seed derivation, BUT the market_id comes from config, not params. The Anchor `seeds` constraint can reference other accounts in the struct. Compute `config.market_counter.checked_add(1).unwrap().to_le_bytes()` in seeds.

    **NOTE on Anchor seed limitations:** Anchor seeds constraints cannot call methods on accounts directly. The standard pattern is to pass the expected market_id as a computed value. Since config.market_counter is read at constraint evaluation time and incremented in the handler, the seed must use the PRE-increment value + 1. Use this pattern:
    ```rust
    seeds = [b"market", config.market_counter.checked_add(1).unwrap().to_le_bytes().as_ref()],
    ```
    If Anchor rejects this (some versions require const seeds), use the alternative: pass `market_id` as a u64 param, validate it equals `config.market_counter + 1` in the handler, and use `#[instruction(params: CreateMarketParams)]` for seed derivation. Adjust params to include `market_id: u64` if needed.

    **Handler validations (in order):**
    1. `let clock = Clock::get()?;` then `let now = clock.unix_timestamp;`
    2. `require!(params.resolution_time > now + 3600, AvenirError::DeadlineTooSoon)` -- 1 hour minimum
    3. `require!(params.category <= 4, AvenirError::InvalidCategory)` -- 0-4 range
    4. `require!(!params.question.is_empty(), AvenirError::EmptyQuestion)`
    5. `require!(params.question.len() <= 200, AvenirError::QuestionTooLong)` -- explicit check
    6. `require!(!params.resolution_source.is_empty(), AvenirError::EmptyResolutionSource)`
