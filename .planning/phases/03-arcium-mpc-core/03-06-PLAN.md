  <name>Task 1: Create GitHub Actions CI workflow for arcium test on x86_64</name>
  <files>.github/workflows/arcium-tests.yml</files>
  <action>
Create `.github/workflows/arcium-tests.yml` that runs all MPC tests on an x86_64 Linux runner.

**Workflow triggers:**
- `push` to `main` and `master` branches (when tests/ or programs/ or encrypted-ixs/ change)
- `pull_request` to `main` (same path filters)
- `workflow_dispatch` for manual triggering

**Runner:** `ubuntu-latest` (x86_64 -- resolves the ARM64 Docker incompatibility)

**Steps:**

1. **Checkout** repository

2. **Install Rust toolchain**
   - Use `dtolnay/rust-toolchain@stable` with `rustc 1.93.0` or latest stable
   - Components: `rustfmt`, `clippy`

3. **Install Solana CLI**
   ```bash
   sh -c "$(curl -sSfL https://release.anza.xyz/stable/install)"
   echo "$HOME/.local/share/solana/install/active_release/bin" >> $GITHUB_PATH
   solana --version
   solana-keygen new --no-bip39-passphrase
   ```

4. **Install Anchor CLI**
   ```bash
   cargo install --git https://github.com/coral-xyz/anchor avm --force
   avm install 0.32.1
   avm use 0.32.1
   anchor --version
   ```

5. **Install Arcium CLI**
   ```bash
   curl -L https://github.com/arcium-hq/arcium-cli/releases/download/v0.8.5/arcium-cli-x86_64-unknown-linux-gnu -o /usr/local/bin/arcium
   chmod +x /usr/local/bin/arcium
   arcium --version
   ```

6. **Install Bun**
   ```bash
   curl -fsSL https://bun.sh/install | bash
   echo "$HOME/.bun/bin" >> $GITHUB_PATH
   ```

7. **Install Node dependencies**
   ```bash
   bun install
   ```

8. **Build Arcium circuits**
   ```bash
   arcium build
   ```

9. **Run arcium test**
   ```bash
   arcium test
   ```
   This runs the Anchor.toml `[scripts] test` command, which executes:
   `bun run ts-mocha -p ./tsconfig.json -t 1000000 "tests/**/*.ts"`

   This will execute:
   - tests/mpc/hello-world.ts (4 tests)
   - tests/mpc/update-pool.ts (4 tests)
   - tests/mpc/client-encryption.ts (4 tests)

**Caching:** Cache Cargo registry and target dir, Solana tools, and node_modules for faster subsequent runs.

**Timeout:** Set job timeout to 30 minutes (MPC tests with local Arx nodes can be slow).

**Important notes:**
- Docker is available by default on `ubuntu-latest` GitHub Actions runners
- The x86_64 runner resolves the ARX node Docker image compatibility issue
- `arcium test` starts a local solana-test-validator with Arx nodes via Docker, then runs the test script
- The `localnet_timeout_secs = 60` in Arcium.toml gives the test validator time to start
- If `arcium test` has the same startup timeout issue observed locally, increase `localnet_timeout_secs` in Arcium.toml or add a `startup_wait` to Anchor.toml
