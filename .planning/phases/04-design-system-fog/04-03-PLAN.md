**Implementation:**

2. In the `MarketDetail` route component function:

   - Find matching market: `const market = MOCK_MARKETS.find(m => m.id === id) ?? MOCK_MARKETS[0]` (fallback to first market for any unmatched ID during development)
3. **Layout structure:**
   ```
   <div className="space-y-6">
     {/* Back link */}
     <Link to="/" className="text-sm text-muted-foreground hover:text-foreground inline-flex items-center gap-1">
       <- Back to Markets
     </Link>

     {/* Sidebar layout: info left, bet form sticky right */}
     <div className="grid gap-6 lg:grid-cols-3">
       <div className="lg:col-span-2">
         <MarketDetail market={market} />
       </div>
       <aside className="lg:sticky lg:top-20 lg:self-start">
         <BetPlacement market={market} />
       </aside>
     </div>
   </div>
   ```
