From encrypted-ixs/src/lib.rs (6 circuits):
- init_pool: Initializes encrypted pool counters (Mxe-owned zeros)
- update_pool: Encrypted bet + encrypted pool -> updated pool + revealed sentiment bucket
- compute_payouts: Encrypted pool -> revealed (yes_total, no_total) at resolution
- init_dispute_tally: Initializes encrypted vote counters
- add_dispute_vote: Encrypted vote + encrypted tally + plaintext stake -> updated encrypted tally
- finalize_dispute: Encrypted tally -> revealed (yes_votes, no_votes)

Key types:
- PoolTotals = [u64; 2] (yes_pool, no_pool)
- BetInput { is_yes: bool, amount: u64 }
- VoteTotals = [u64; 2] (yes_weighted, no_weighted)
- VoteInput { is_yes: bool }

Encryption types used:
- Enc<Shared, T>: User-encrypted input (x25519 shared secret)
- Enc<Mxe, T>: MPC-network-owned encrypted state (stored on-chain)
- .reveal(): Selective declassification (sentiment bucket, payout totals)

From programs/avenir/src/state/:
- Config: Protocol settings (admin, fees, USDC mint)
- Market: Question, deadline, state, sentiment, MPC lock
- MarketPool: Encrypted ciphertext (yes/no pool totals + nonce)
- UserPosition: Per-user bet amounts + claimed flag
- CreatorWhitelist: Market creator authorization

On-chain account model:
- Market PDA: ["market", market_id] -- plaintext metadata
- MarketPool PDA: ["market_pool", market_id] -- encrypted ciphertext
- UserPosition PDA: ["position", market_id, user] -- plaintext amounts
- DisputeTally PDA: encrypted vote ciphertext

2. **The Problem (hook)**: 1-2 paragraphs about the herding problem in prediction markets. Key points:
   - On platforms like Polymarket, seeing which side is winning causes crowd-following behavior
   - Whales can manipulate visible odds to influence betting patterns
   - The signal that makes prediction markets valuable (aggregated independent beliefs) gets corrupted when participants can see and react to each other's positions
   - This is the "Keynesian beauty contest" problem applied to prediction markets

   - At resolution, pool totals are decrypted and winners receive proportional USDC payouts
